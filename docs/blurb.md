Adds a Google Street View preview to the RideWithGPS route planner/viewer. Hovering over a route shows the road ahead.

* ...

The map alone doesn't show road conditions — surface, shoulder width, or traffic. This extension adds a Street View preview on hover, oriented in the direction the route is heading.

- Hover near any part of a route to see a Street View preview
- Shows street name and city, plus a compass and heading
- Click the preview to open full Google Maps Street View
- Works in both the route editor and route viewer, on any public route

Requirements:
* Google Maps selected as the map type in RideWithGPS
* Google Maps API key with Street View Static API enabled (free tier: ~28,500 requests/month). Setup instructions included.

Fully open source under MIT license: https://github.com/nslussar/rwgps-streetview
