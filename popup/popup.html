Street View Preview

How to get an API key
  1. Go to Google Cloud Console
  2. Create a project (or select an existing one)
  3. Enable the Street View Static API
  4. Go to Credentials → Create Credentials → API Key
  5. Optionally restrict the key to the Street View Static API
  6. Paste the key above

The Street View Static API has a free tier of $200/month credit (~28,500 requests).
